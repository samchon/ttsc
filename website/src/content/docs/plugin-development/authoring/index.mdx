# Authoring

Five chapters that take you from `npm init` to a publishable plugin.

1. [Getting started](/docs/plugin-development/authoring/getting-started) — build the smallest useful source-transform plugin end to end.
2. [Recipes](/docs/plugin-development/authoring/recipes) — focused patterns you can copy.
3. [Testing](/docs/plugin-development/authoring/testing) — Go unit tests + end-to-end `ttsc` fixtures.
4. [Publishing](/docs/plugin-development/authoring/publishing) — npm package shape and pre-publish checks.
5. [Local development](/docs/plugin-development/authoring/local-dev) — `go.work`, `gopls`, `go test`, pnpm notes.

Before you start, skim [Concepts](/docs/plugin-development/concepts). After you ship, see [Reference](/docs/plugin-development/reference).
